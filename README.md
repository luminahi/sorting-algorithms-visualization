## Installation

- Clone the repository
   ```bash
   https://github.com/luminahi/sorting-algorithms-visualization-canvas.git
   ```
- Install dependencies
   ```bash
   npm install
   ```
- Generate Javascript
  ```bash
  npm run build 
  ```

## Usage

- Run the application
   ```bash
   npm start
   ```
   the page will open in localhost, port 8000 by default.
